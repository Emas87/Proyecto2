# Proyecto2

Compilar todo:
make all

Borrar ejecutables:
make clean

Correr Creador:
./Creador -b  <nombre buffer> -s <tamano del buffer>

Correr Productor:
./Productor -b  <nombre buffer> -t <media en segundos>

Correr Consumidor:
./Consumidor -b  <nombre buffer>-t <media en segundos>

Correr Finalizador:
./Finalizador -b <nombre buffer>
